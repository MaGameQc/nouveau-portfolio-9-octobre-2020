BurgerMaker

C'est une petite application web qui vous permet de choisir l'animation de votre hamburger icon préféré, copier son code HTML et CSS, puis le coller dans votre propre projet.

C'est Quoi BurgerMaker ?

en tant que développeur, une des choses que j'aime bien faire sont les hamburger icons. ce fameux bouton qui est toujours là à nous attendre si on va sur un site web avec un appareille mobile. Puisque je cherchais un projet qui me ferait pratiquer Javascript, je me suis dit, "bah je devrais coder une liste de ces icônes, tu choisie celle que tu veut sur le site, et ca te génère le code HTML et CSS afin de pouvoir t'en servir dans tes projets". Ce projet m'en à appris plus sur la programmation orientée objet, qu'il est important d'encapsuler, de séparer son code en plusieurs methodes, sinon, c'est très difficile de s'y retrouver.
